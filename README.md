# bidding-platform

## 项目结构

- frontend: 前端项目
- backend: 后端项目

## 运行

- 运行后端：`run-backend.cmd`
- 运行前端：`run-frontend.cmd`

## 部署

- 使用 GitHub Actions 部署到 DigitalOcean Droplet

## 开发

- 使用 Angular CLI 开发前端
- 使用 Nest.js 开发后端

## 参考

- [Nest.js 文档](https://docs.nestjs.com/)
- [Angular 文档](https://angular.io/docs)
